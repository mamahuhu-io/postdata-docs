Postdata Desktop App is a cross-platform desktop application that helps you create and manage API requests. It is built on top of the [Postdata Web Client](/documentation/clients/web) and is powered by [Tauri](https://tauri.app).

## Install Postdata Desktop App

### macOS

1. Download the latest version of Postdata Desktop App from the [Postdata website](https://postdata.cn/download).
2. Open the downloaded file.
3. Drag and drop Postdata Desktop App to the Applications folder.
4. Open Postdata Desktop App from the Applications folder.
5. If you see a warning message, click "**Open**".

### Windows

1. Download the latest version of Postdata Desktop App from the [Postdata website](https://postdata.cn/download).
2. Open the downloaded file.
3. Follow the on-screen instructions to install Postdata Desktop App.

### Linux

1. Download the latest version of Postdata Desktop App from the [Postdata website](https://postdata.cn/download).
2. Open the downloaded file.
3. Follow the on-screen instructions to install Postdata Desktop App.
4. If you see a warning message, click "**Open**".
