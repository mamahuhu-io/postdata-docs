To get started with Postdata, you need to choose a platform to use it on. Postdata is available on the following platforms:

- [Desktop](#desktop-app)
- [CLI](#cli)


## Desktop App

Postdata Desktop App is a cross-platform desktop application built with Tauri and Postdata Web Client. It is a standalone version of Postdata that can be installed on your computer and used without a browser. Postdata Desktop App is available for Mac, Windows and Linux.

  <Card title="Download Desktop App" iconType="light" icon="circle-arrow-right" href="https://postdata.cn/download"></Card>
</CardGroup>

## CLI

Postdata CLI is the command-line interface for Postdata. It is a standalone version of Postdata that can be installed on your computer and used without a browser. Postdata CLI is available as an npm package.
